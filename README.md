# Secure Notes (IT Security Project)

A console application built in Java to securely store notes. 
This project was created for an IT Security assignment to demonstrate secure password handling and basic database operations.

## Features
* User Registration and Login
* Passwords are securely hashed using BCrypt before saving to the database
* Users can create, view, edit, and delete their own notes
* SQLite database (`securenotes.db`) is used for storage
* Admin role can view all notes in the system and delete any note
* Prepared Statements are used to protect against SQL Injection
